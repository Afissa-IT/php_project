# Community Tool Library - PHP/MySQL CRUD

## Setup (local)
1. Import `db.sql` into your MySQL server (via phpMyAdmin, MySQL Workbench, or `mysql < db.sql`).
2. Edit `config.php` and set your DB credentials if needed.
3. Put the project in your local server folder (XAMPP `htdocs` or WAMP `www`).
4. Visit `http://localhost/community_tool_library/index.php`.
